
# Blueprint: 드론 사진 홍보 사이트

## 개요

이 프로젝트는 드론으로 촬영한 사진을 홍보하고 잠재 고객에게 서비스를 알리는 것을 목표로 하는 웹사이트입니다. 이 문서는 프로젝트의 초기 버전부터 현재까지의 모든 스타일, 디자인, 기능 구현에 대한 세부 정보를 포함합니다.

## 초기 계획 (v1)

### 디자인 및 스타일

*   **색상 팔레트:** 전문성과 기술적 느낌을 강조하기 위해 어두운 테마를 기반으로 하며, 포인트 색상으로 밝은 청록색 또는 주황색을 사용합니다.
*   **타이포그래피:** 가독성이 좋은 모던한 산세리프 폰트를 사용합니다. (예: Noto Sans KR, Pretendard)
*   **레이아웃:** 사용자의 시선을 사로잡는 대형 히어로 이미지와 함께 깔끔하고 직관적인 섹션 구성을 가집니다.
*   **시각 효과:**
    *   메인 배경에 미세한 노이즈 텍스처를 추가하여 고급스러운 느낌을 줍니다.
    *   카드 및 버튼에 부드러운 그림자 효과를 적용하여 입체감을 더합니다.
    *   갤러리 이미지에 마우스를 올렸을 때 미묘한 확대 효과를 추가합니다.

### 기능

*   **헤더:** 로고와 함께 '소개', '갤러리', '서비스', '문의하기' 등의 메뉴를 포함한 탐색 바.
*   **히어로 섹션:** 시선을 사로잡는 드론 영상 또는 이미지 배경과 함께 서비스의 핵심 메시지를 전달하는 헤드라인.
*   **갤러리 섹션:** 다양한 드론 사진을 전시하는 반응형 이미지 갤러리.
*   **서비스 섹션:** 제공하는 서비스(예: 항공 촬영, 영상 제작, 데이터 분석 등)를 아이콘과 함께 설명하는 카드 형태의 레이아웃.
*   **문의하기 섹션:** 사용자가 쉽게 문의를 남길 수 있는 간단한 폼.
*   **푸터:** 저작권 정보, 소셜 미디어 링크 등을 포함.

### 기술 스택

*   **HTML:** 시맨틱 마크업을 사용하여 구조를 정의합니다.
*   **CSS:** 모던 CSS 기능(변수, Flexbox, Grid)을 사용하여 스타일과 반응형 디자인을 구현합니다.
*   **JavaScript:** 간단한 상호작용 및 갤러리 기능을 위해 ES 모듈을 사용합니다.
